# 离线缓存功能测试指南

## 🎯 测试目标

验证 LibreTV 的离线缓存功能是否正常工作，包括：
- 离线管理页面能否正常打开
- 离线缓存列表能否正常显示
- 离线视频能否正常播放

## 📋 前置条件

1. **确保已更新到最新版本**（v3.2.0 或更高）
2. **清除浏览器缓存**（特别是 Service Worker）
3. **清除 IndexedDB 数据**（重要！）

## 🔧 清除 IndexedDB 数据

### 方法 1：通过开发者工具（推荐）

1. 按 `F12` 打开开发者工具
2. 进入 "Application" 或 "应用程序" 标签
3. 左侧找到 "Storage" > "IndexedDB"
4. 找到 "LibreTVOffline" 数据库
5. 右键点击并选择 "Delete database" 或 "删除数据库"
6. 刷新页面

### 方法 2：通过浏览器设置

1. 打开浏览器设置
2. 进入 "隐私和安全" > "网站设置"
3. 找到 "LibreTV" 相关的网站设置
4. 清除 "存储" 或 "IndexedDB" 数据

## 🧪 测试步骤

### 第一步：测试离线管理页面

1. **访问首页**：
   - 打开 LibreTV 首页
   - 确认页面正常加载

2. **点击离线缓存按钮**：
   - 点击左上角的缓存图标（下载图标）
   - 观察是否能在新标签页中打开 `offline.html`

3. **检查页面显示**：
   - 应该能看到 "离线缓存" 标题
   - 应该能看到 "暂无离线缓存" 的提示（如果没有缓存的话）
   - 页面不应该有错误

**预期结果**：
- ✅ 离线管理页面能正常打开
- ✅ 页面显示正常，没有错误
- ✅ 如果没有缓存，显示 "暂无离线缓存"

### 第二步：测试离线缓存功能

1. **搜索视频**：
   - 在首页搜索框输入关键词
   - 选择一个视频

2. **开始缓存视频**：
   - 点击视频的播放按钮
   - 进入播放器页面
   - 点击播放器控制栏的 "离线" 按钮
   - 开始缓存视频

3. **观察缓存进度**：
   - 应该能看到缓存进度条
   - 应该能看到缓存状态（缓存中/已完成）
   - 等待缓存完成

**预期结果**：
- ✅ 能看到缓存进度条
- ✅ 缓存状态正常显示
- ✅ 缓存能正常完成

### 第三步：测试离线管理列表

1. **返回离线管理页面**：
   - 返回首页
   - 点击左上角的缓存图标
   - 打开离线管理页面

2. **检查缓存列表**：
   - 应该能看到刚才缓存的视频
   - 应该能看到视频的标题、集数、大小等信息
   - 应该能看到 "播放" 按钮

**预期结果**：
- ✅ 能看到缓存的视频列表
- ✅ 视频信息显示正确
- ✅ 能看到播放按钮

### 第四步：测试离线播放

1. **播放离线视频**：
   - 在离线管理页面中
   - 点击某个视频的 "播放" 按钮
   - 观察是否能正常跳转到播放器

2. **检查播放状态**：
   - 播放器应该正常加载
   - 视频应该能正常播放
   - 不需要网络连接

**预期结果**：
- ✅ 能正常跳转到播放器
- ✅ 视频能正常播放
- ✅ 播放器显示正常

### 第五步：测试离线删除

1. **删除离线视频**：
   - 返回离线管理页面
   - 点击某个视频的 "删除" 按钮
   - 确认删除

2. **检查删除结果**：
   - 视频应该从列表中消失
   - 存储空间应该减少

**预期结果**：
- ✅ 视频从列表中消失
- ✅ 存储空间正确更新

## 🔍 调试信息

如果测试失败，请检查以下信息：

### 打开浏览器控制台

1. 按 `F12` 打开开发者工具
2. 进入 "Console" 或 "控制台" 标签

### 关键日志信息

查找以下关键日志：

```
[Offline] Opening offline list page
```

### 常见错误信息

1. **数据库版本错误**：
   ```
   Error: Version error
   ```
   **解决方案**：清除 IndexedDB 数据后重试

2. **数据库被占用**：
   ```
   Error: 数据库被占用
   ```
   **解决方案**：关闭其他标签页后重试

3. **存储空间不足**：
   ```
   Error: QuotaExceededError
   ```
   **解决方案**：删除一些离线视频或清理浏览器存储

## 🚨 常见问题

### Q1: 离线管理页面打不开

**解决方案**：
1. 检查浏览器控制台是否有错误
2. 确保已清除 IndexedDB 数据
3. 尝试强制刷新页面（Ctrl + Shift + R）

### Q2: 缓存列表显示为空

**解决方案**：
1. 确认是否真的有缓存视频
2. 检查 IndexedDB 数据是否正确
3. 尝试重新缓存视频

### Q3: 离线视频无法播放

**解决方案**：
1. 检查缓存是否完整
2. 检查 IndexedDB 数据是否正确
3. 尝试重新缓存视频

### Q4: 缓存进度一直显示 0%

**解决方案**：
1. 检查网络连接
2. 检查视频源是否可用
3. 尝试其他视频源
